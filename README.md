- Chủ đề: Thực hành ngoại vi SPI trên STM32
- Chức năng: Gửi dữ liệu từ SPI1 sang SPI2 trên STM32
- Cách sử dụng:
1. Mở file cấu hình (.ioc) để xem cấu hình STM32, Nối các chân đúng cấu hình.
2. Gắn dây nạp STM32.
3. Mở file project trên STM32CubeIDE để compile và chạy chương trình.
4. Trong chế độ Debug, gán biến trạng thái và biến buffer để giám sát quá trình gửi dữ liệu .
